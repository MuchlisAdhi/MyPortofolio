## Proyek Unggulan

### ProductSAF_v2 (PWA)

Salah satu proyek yang saya kembangkan adalah sistem berbasis web yang dapat diinstal seperti aplikasi mobile (Progressive Web App).

Repo: https://github.com/MuchlisAdhi/ProductSAF_v2

Highlight:

- Implementasi **Progressive Web App (PWA)**
- Bisa diinstal di device (mobile/desktop)
- Fokus pada performa dan usability
- Digunakan untuk kebutuhan sistem internal

Proyek ini menunjukkan bagaimana web app bisa memberikan pengalaman seperti native app tanpa harus publish ke store.
